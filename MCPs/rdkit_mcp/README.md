# RDKit MCP

**Tipo**: MCP Python locale (subprocess)

## Funzione
Permette a Prismalux di eseguire chimica computazionale: disegnare molecole,
calcolare proprietà (MW, LogP, TPSA), generare fingerprint, convertire SMILES→SDF.

## Installazione
```bash
pip install rdkit
```
