# Anki MCP (AnkiConnect)

**Porta**: 8765  
**Tipo**: Plugin Anki preesistente (AnkiConnect)

## Funzione
Permette a Prismalux di creare flashcard, aggiungere note, gestire mazzi Anki
direttamente dall'AI senza aprire Anki manualmente.

## Installazione
1. Apri Anki → Tools → Add-ons → Get Add-ons
2. Codice: **2055492159** (AnkiConnect ufficiale)
3. Riavvia Anki — server HTTP parte automaticamente su porta 8765

## Uso da Prismalux
```
APP Controller → Anki → crea flashcard da testo/AI
```
