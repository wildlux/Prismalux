# 📊 Tipologie di Grafici Esistenti

Ecco una lista completa delle principali tipologie di grafici, organizzate per categoria:

---

## 🔹 Grafici di Confronto
| Grafico | Utilizzo principale |
|---------|-------------------|
| **Barre verticali (Column)** | Confrontare valori tra categorie |
| **Barre orizzontali** | Confrontare categorie con etichette lunghe |
| **Barre raggruppate** | Confrontare più serie di dati per categoria |
| **Barre impilate** | Mostrare composizione e confronto simultaneamente |
| **Grafico a imbuto (Funnel)** | Visualizzare processi a stadi (es. vendite) |

---

## 🔹 Grafici di Composizione/Parte-tutto
| Grafico | Utilizzo principale |
|---------|-------------------|
| **Torta (Pie chart)** | Mostrare percentuali di un totale (max 5-6 categorie) |
| **Ciambella (Donut)** | Variante della torta con spazio centrale |
| **Treemap** | Visualizzare gerarchie e proporzioni con rettangoli |
| **Sunburst** | Gerarchie circolari annidate |
| **Grafico a cascata (Waterfall)** | Mostrare contributi cumulativi a un totale |
| **Grafico a pile 100%** | Confrontare composizioni percentuali tra gruppi |

---

## 🔹 Grafici di Distribuzione
| Grafico | Utilizzo principale |
|---------|-------------------|
| **Istogramma** | Distribuzione di frequenza di dati continui |
| **Box plot (Box-and-whisker)** | Median, quartili e outlier |
| **Violin plot** | Distribuzione + densità dei dati |
| **Dot plot** | Distribuzione di piccoli dataset |
| **Grafico a densità (Density plot)** | Curva di probabilità continua |

---

## 🔹 Grafici di Relazione/Correlazione
| Grafico | Utilizzo principale |
|---------|-------------------|
| **Scatter plot (Dispersione)** | Relazione tra due variabili numeriche |
| **Bubble chart** | Scatter plot con terza dimensione (dimensione bolla) |
| **Heatmap** | Intensità di valori in matrice 2D |
| **Grafico a bolle con colore** | Quattro dimensioni: x, y, dimensione, colore |

---

## 🔹 Grafici Temporali/Serie Storiche
| Grafico | Utilizzo principale |
|---------|-------------------|
| **Linea (Line chart)** | Trend nel tempo |
| **Area** | Volume cumulativo nel tempo |
| **Area impilata** | Composizione temporale |
| **Grafico a candele (Candlestick)** | Dati finanziari (open/high/low/close) |
| **Grafico OHLC** | Variante semplificata delle candele |

---

## 🔹 Grafici Geospaziali
| Grafico | Utilizzo principale |
|---------|-------------------|
| **Mappa coropletica** | Valori per aree geografiche (colorate) |
| **Mappa a bolle** | Punti geografici con dimensione proporzionale |
| **Mappa di flusso (Flow map)** | Spostamenti tra località |
| **Cartogramma** | Aree distorte in base a variabile statistica |

---

## 🔹 Grafici Gerarchici e di Rete
| Grafico | Utilizzo principale |
|---------|-------------------|
| **Albero (Tree diagram)** | Strutture gerarchiche |
| **Grafico a radiale (Radial tree)** | Gerarchia circolare |
| **Grafo/Network chart** | Relazioni tra nodi (social network, ecc.) |
| **Diagramma di Sankey** | Flussi di quantità tra nodi |

---

## 🔹 Grafici Specializzati/Avanzati
| Grafico | Utilizzo principale |
|---------|-------------------|
| **Radar chart (Spider)** | Confronto multivariato su assi radiali |
| **Grafico a parallele coordinate** | Dati multidimensionali |
| **Grafico a dispersione 3D** | Relazione tra tre variabili numeriche |
| **Grafico a termometro** | KPI rispetto a un target |
| **Gauge chart** | Valore su scala semicircolare (es. velocità) |
| **Bullet graph** | Performance vs target con contesto |
| **Grafico a Gantt** | Pianificazione temporale di attività |
| **Grafico a piramide** | Confronto simmetrico (es. demografia) |
| **Word cloud** | Frequenza termini in testo |
| **Grafico a corda (Chord diagram)** | Relazioni bidirezionali tra entità |

---

## 🔹 Grafici Interattivi/Moderni
| Grafico | Utilizzo principale |
|---------|-------------------|
| **Grafico animato** | Mostrare evoluzione temporale dinamica |
| **Small multiples** | Serie di grafici simili per confronto |
| **Dashboard combinate** | Più visualizzazioni integrate |
| **Grafici con tooltip/drill-down** | Esplorazione interattiva dei dati |

---

## ✅ Come Scegliere il Grafico Giusto?

```mermaid
graph LR
    A[Cosa vuoi mostrare?] --> B[Confronto]
    A --> C[Composizione]
    A --> D[Distribuzione]
    A --> E[Relazione]
    A --> F[Trend temporale]
    A --> G[Geolocalizzazione]

    B --> H[Barre/Colonne]
    C --> I[Torta/Treemap/Waterfall]
    D --> J[Istogramma/Box plot]
    E --> K[Scatter/Heatmap]
    F --> L[Linea/Area]
    G --> M[Mappa coropletica]
```

---

> 💡 **Consigli pratici**:
> - Usa massimo 5-7 categorie nei grafici a torta
> - Preferisci barre orizzontali per etichette lunghe
> - Evita il 3D decorativo: distorce la percezione
> - Scegli colori accessibili (daltonismo-friendly)
> - Mantieni la semplicità: meno è spesso di più 🎯
