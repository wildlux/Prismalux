COME USARE QUESTA CARTELLA
===========================

1. Metti qui il tuo Curriculum Vitae.

2. Formati supportati:
   - .txt  (testo semplice, funziona sempre)
   - .pdf  (richiede: pip install pypdf)

3. Il nome del file non importa. Esempi validi:
   - mio_cv.txt
   - curriculum_vitae.pdf
   - CV_Mario_Rossi.txt

4. Se metti più file, viene usato il più recente.

5. Il programma leggerà il CV, lo analizzerà con l'AI
   e ti proporrà i ruoli più adatti da cercare —
   così non devi riscrivere ogni volta cosa sei.

CONSIGLIO
---------
Se hai il CV in Word (.docx), salvalo come .txt con:
  File → Salva con nome → Formato: Testo normale (.txt)
